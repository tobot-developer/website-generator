# AI Website Generator v4.0

A professional, AI-powered website generator with dark luxury design.

## 🚀 Live Demo

**URL:** https://website-generator-h6bu1paas-tobot-s-projects.vercel.app

## ✨ Features

- **Dark Luxury Design** - Premium aesthetics with gold accents
- **Complete Form** - Company info, tagline, description, industry
- **Design Preferences** - Dark/Light/Modern/Bold styles
- **Color Palettes** - Gold, Ocean Blue, Emerald, Royal Purple, Rose
- **Feature Selection** - Hero, About, Services, Stats, Testimonials, Team, Portfolio, Pricing, FAQ, Contact, Blog, Newsletter
- **Contact Info** - Email, phone, location
- **Advanced Options** - Backend API toggle, responsive design, SEO optimization
- **Live Preview** - Modal preview of generated website
- **ZIP Download** - Download complete website package with README

## 🛠 Tech Stack

- Pure HTML/CSS/JS (no frameworks needed)
- JSZip for ZIP generation
- FileSaver.js for downloads
- Cormorant Garamond + DM Sans fonts
- Grain texture animation

## 📦 Deployment

Deployed on Vercel with automatic CI/CD from GitHub.

## 📝 Usage

1. Fill in the form with your company details
2. Select design preferences and features
3. Click "Generate Website"
4. Preview or download your website
